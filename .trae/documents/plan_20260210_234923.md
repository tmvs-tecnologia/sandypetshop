Vou desenvolver um novo **Painel de Estatísticas Unificado (Dashboard)** para substituir e aprimorar os modais existentes.

**Estrutura do Componente:**
1.  **`StatisticsDashboardModal.tsx`**: Componente principal com abas de navegação.
    *   **Visão Geral:** KPIs em tempo real (Hoje, Semana, Mês) unificando dados de Loja e Pet Móvel.
    *   **Financeiro:** Gráficos de receita (Barras/Linhas SVG) com comparativo anual (2025 vs 2026) e detalhamento de ganhos.
    *   **Operacional:** Distribuição de serviços (Pizza SVG) e horários de pico.
    *   **Mensalistas:** Seção dedicada com métricas de receita recorrente e status de pagamentos.

**Integração de Dados (Supabase):**
*   Busca unificada nas tabelas `appointments` e `pet_movel_appointments` para calcular totais de serviços avulsos.
*   Integração com `monthly_clients` para projetar receita recorrente.
*   Cache local (`useMemo`/`useState`) para performance.

**Design e UX:**
*   Layout responsivo com Grid/Flexbox.
*   Visual "Glassmorphism" sutil e cores da marca (Pink/Purple).
*   Animações de entrada e transição de abas.
*   Gráficos SVG customizados (sem dependências externas pesadas) para "High Craft".

**Passos de Implementação:**
1.  Criar o componente `StatisticsDashboardModal.tsx`.
2.  Implementar a lógica de busca e agregação de dados.
3.  Desenvolver os componentes visuais dos gráficos.
4.  Adicionar botão de acesso no `App.tsx` (substituindo ou complementando os atuais).
