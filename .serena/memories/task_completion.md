# Critérios de Finalização de Tarefas

1. **Verificação de Build**: Certifique-se de que o código compila perfeitamente executando `npm run build`.
2. **Commit no Git**: Escreva os comentários de commit no git sempre em **Português do Brasil**.
3. **Deploy**: O deploy final é verificado e construído na Vercel (sistema de build padrão).
