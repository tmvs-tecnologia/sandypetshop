# Tech Stack do Projeto

## Frontend
- **React**: ^19.1.1 (React DOM ^19.1.1)
- **TypeScript**: ~5.8.2
- **Vite**: ^6.2.0 (Bundler e Dev Server)
- **Estilização**: CSS Vanilla (prioridade para estética rica, moderna e premium; gradients, glassmorphism e micro-animações). Evitar TailwindCSS a menos que seja explicitamente solicitado.
- **Iconografia**: `lucide-react` e `@heroicons/react`

## Backend & Database
- **Supabase**: `@supabase/supabase-js` (^2.78.0) para banco de dados e autenticação

## Build & Deploy
- **Vercel**: Sistema de build e deploy oficial do projeto
