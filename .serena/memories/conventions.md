# Convenções do Projeto

## Geral
- **Idioma**: Respostas ao usuário e comentários de commits no git devem ser escritos sempre em **Português do Brasil**.
- **Aesthetic**: Design rich aesthetics (uso de gradients, glassmorphism, Outfit/Inter typography e micro-animações). Não usar cores genéricas.
- **Skills**: Sempre que for solicitado realizar correções no frontend, utilize as skills `frontend-design` e `frontend-developer`.

## Git
- Sempre escreva os comentários do push/commit no git em **Português do Brasil**.
