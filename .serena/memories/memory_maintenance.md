# Convenções de Manutenção de Memória

Este projeto usa Serena para manter a documentação da arquitetura e as convenções do código atualizadas.

## Regras de Atualização
- As memórias devem ser mantidas concisas, limpas e em Português do Brasil.
- Sempre que houver uma alteração significativa no stack tecnológico ou em convenções cruciais, atualize a respectiva memória.
- Mantenha os nomes dos arquivos coerentes e referencie outras memórias usando o prefixo `mem:`.
