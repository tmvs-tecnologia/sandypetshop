# Comandos Sugeridos

## Ambiente Local (Windows)
- Instalar dependências:
  ```bash
  npm install
  ```
- Executar servidor de desenvolvimento local:
  ```bash
  npm run dev
  ```
- Executar o build local:
  ```bash
  npm run build
  ```
- Visualizar o build localmente:
  ```bash
  npm run preview
  ```

## Testes Automatizados
- Executar testes com Playwright:
  ```bash
  npx playwright test
  ```
