# Best Skills - Recommended Skills

Esta configuração define as skills recomendadas para o projeto Sandy PetShop.

## Skills Preferenciais

Para tarefas de frontend (React, Next.js, UI), use:
- **frontend-developer** - Para implementação de componentes React/Next.js
- **frontend-design** - Para design de interfaces distintas e não-genéricas
- **frontend-slides** - Para criar apresentações HTML

Para tarefas de dados/SQL:
- **sql-pro** - Para queries SQL avançadas e otimização

Para tarefas de debugging:
- **systematic-debugging** - Sempre use antes de propor correções de bugs

Para tarefas de IA/LLM:
- **ai-engineer** - Para implementação de features de IA

Para tarefas de automação Supabase:
- **supabase-automation** - Para operações no banco Supabase

Para tarefas de marketing:
- **marketing-ideas** - Para estratégias de marketing com scoring de viabilidade

Para dashboards:
- **kpi-dashboard-design** - Para design de dashboards KPI

## Como Usar

Em seus pedidos, você pode simplesmente mencionar qual skill usar:
- "Crie um componente de login usando a skill frontend-developer"
- "Há um bug no login, use a skill systematic-debugging para investigar"
